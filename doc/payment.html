電子化支付

  • 行動支付
  • 第三方支付
  • 電子支付
  • 電子票證

行動支付

所有透過智慧型手機支付的行為，就是行動支付。

  • NFC 感應付款：數位版信用卡、Debit Card、悠遊卡等綁進卡匣
    • Apple Pay / Google Pay / Samsung Pay
  • QR Code 掃碼支付：出示或掃描店家 QR Code
    • LINE Pay / 街口支付 / Pi 錢包

第三方支付

  • 經濟部管轄的業務，代理收付交易。
  • 為保障買家及賣家之間的交易安全，錢會先寄放在第三方支付單位的帳戶中，確認到貨後再由第三方將錢交給賣家。
    • Paypal / 藍新金流 / 綠界 ECPay / OPEN 錢包

電子支付

  • 由金管會管理
  • 比第三方支付多了轉帳及儲值功能。
    • LINE Pay / LINE Pay 一卡通 / 街口支付 / Pi 錢包

電子支付業務

  • 22 家金融機構：

    臺銀、土銀、合庫、一銀、華銀、彰銀、富邦、國泰世華、兆豐、花旗、臺灣中小、台中銀行、新光、陽信、中華郵政、元大、永豐、玉山、台新、中信、遠東、上海

  • 4 家電子票證公司：

    悠遊卡、一卡通、愛金卡、遠鑫票證

  • 5 家專營電子支付機構：

    歐付寶電子支付股份有限公司、橘子支行動支付股份有限公司、簡單行動支付股份有限公司、國際連股份有限公司、街口支付電子支付股份有限公司

電子票證

  • 主管機關為金管會
  • 實體感應式卡片，可儲值、消費扣款，不可轉帳。
    • 悠遊卡 / 一卡通 / iCash
